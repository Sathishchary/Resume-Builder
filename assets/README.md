# Resume Builder - Assets Directory
This directory contains images, icons, and other static assets for the Resume Builder application.

## Structure
- `images/` - Template previews, logos, and other images
- `icons/` - Favicon and application icons

## Templates
In a real implementation, this would contain:
- Template preview images (PNG/JPG)
- Company logos
- User profile pictures
- Background images

## Icons
- favicon.ico - Website favicon
- app icons for mobile/desktop versions
