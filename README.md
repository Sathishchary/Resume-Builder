# AI-Powered Resume Builder

This project is an AI-powered application to build and analyze resumes. It leverages NLP and machine learning to create, parse, and evaluate professional resumes with intelligent suggestions and formatting.

### Features
- Create professional resumes with AI-powered suggestions
- Multiple template options for different industries
- Automatic formatting and layout optimization
- Skills and experience enhancement suggestions
- Export to PDF and DOCX formats
- Resume scoring and improvement recommendations
- ATS (Applicant Tracking System) compatibility checker

### Tech Stack
- **Backend:** Python, FastAPI
- **NLP/AI:** spaCy, scikit-learn
- **Frontend:** HTML, CSS, JavaScript (or React)
- **Containerization:** Docker

### Setup Instructions
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sathishchary/Resume-Builder.git
   cd Resume-Builder
   ```
2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Run the application:**
   ```bash
   uvicorn main:app --reload
   ```
4. **Access the frontend:**
   Open your browser at [http://localhost:8000](http://localhost:8000)

### Usage
- Create a new resume or import existing one
- Fill in your details with AI-powered suggestions
- Choose from available templates
- Get real-time feedback and improvements
- Export your polished resume
